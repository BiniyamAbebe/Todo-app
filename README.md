# To-Do App with Reminders ✅

A simple, clean to-do list app built with HTML, CSS, and JavaScript that allows users to:
- Add tasks 📝
- Mark them as complete ✅
- Delete tasks ❌
- Set reminders ⏰ (if applicable)

## 🚀 Features
- Minimal UI
- Responsive design
- Easy-to-use task manager
